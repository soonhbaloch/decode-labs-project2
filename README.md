## 📌 Project Overview
As an intern at **DecodeLabs**, this project marks my **Adaptability Phase**. The goal was to move beyond "fixed dimensions" and master the art of **Fluidity**. I have built a professional Coffee Cafe layout that gracefully transitions from desktop monitors to mobile screens using pure responsive logic. 

## 🎯 Requirements Fulfilled
Following the Industrial Training Kit guidelines, I have implemented: 
**CSS Media Queries:** To ensure the design works perfectly across different screen size.
*Responsive Navigation:** A menu system that adapts to mobile and desktop views. 
**Mobile-First Strategy:** Starting with a "Mobile Core" (Base CSS) and adding complexity as space increases. 
**Mandatory Viewport Tag:** Used the `<meta name="viewport">` tag to ensure proper rendering on mobile devices.

## 🛠️ Key Skills Demonstrated
**Fluid Layouts:** Implementing the "Content is Like Water" philosophy. 
  **Flexbox Specialist:** Using `display: flex` for one-dimensional alignment and spacing. 
  **Breakpoint Management:** Setting logical bridges at 480px, 768px, and 1024px.
